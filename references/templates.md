# Self-Introduction Templates by Role Type

All templates follow: **Background → Capabilities → Motivation**.

---

```
[动机]
我一直关注贵公司在[具体产品/技术方向]上的投入。了解到团队正在[JD提到的方向]，这也是我下一阶段的核心目标，非常期待能加入您的团队。
```


## Career Switcher / 转行者

### 1 minute (ZH)

```
[背景]
面试官好，我是[Name]。我的背景可能跟其他候选人不太一样——之前做了[N]年的[原行业/岗位]，[时间]前决定转[新方向]。

[动机]
选择[Target Company]是因为[方向/产品]跟我的复合背景正好契合。我既能[新技能]，也懂[旧行业的业务逻辑]，期待能成为团队里视角最不一样的那个人。
```


---

## Quick Checklist

Before using any template, verify:
1. **Every bracket replaced** with a real, specific detail (not a vague category)
2. **At least one number** in the Capabilities section
3. **Motivation references something real** — a product, a tech choice, a team initiative
4. **Read it aloud** — if it sounds like reading a script, rewrite until it doesn't
